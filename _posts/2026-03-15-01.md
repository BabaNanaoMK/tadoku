# AI音声入力ツールの概観と、最後にSpokenlyをおすすめしたい話

AI音声入力ツールは、この1〜2年でかなり実用段階に入ってきました。  
以前の音声入力は「話した通りにそのまま文字になるだけ」でしたが、最近のツールはそこから一歩進んで、**句読点の補完・文章の整形・文脈に合わせた書き換え**までやってくれます。  
つまり、ただの「音声認識」ではなく、**“書くための音声入力”**になってきています。

この記事では、まず代表的なAI音声入力ツールをざっと整理し、そのうえで**無料で始めやすいSpokenly**を簡単に紹介します。

## 代表的なAI音声入力ツールまとめ

### 1. Apple純正 Dictation
- **料金**：無料（macOS標準機能）
- **特徴**：
  - Macに最初から入っている
  - どこでもすぐ使える
  - 設定が簡単
  - 一部の処理はデバイス上で行われる場合がある
- **評判**：
  - とにかく手軽
  - ただし、最近のAI音声入力アプリと比べると、精度・整形力・多機能さでは見劣りしやすいという評価が多いです。

### 2. Spokenly
- **料金**：
  - ローカルモデル利用は**無料・無制限**
  - 自分のAPIキー持ち込み（BYOK）も無料
  - APIキー不要のクラウド機能込みプランは **$9.99/月**
- **特徴**：
  - Mac / iPhone対応
  - Whisper・Parakeetの**ローカルモデルを無料で使える**
  - **オフライン利用可能**
  - 100以上の言語対応
  - Local Only Modeあり
  - 音声入力だけでなく、AIプロンプトやMac操作向けのAgent Modeもある
- **評判**：
  - 公式サイトでは**App Store評価4.9**、**10万+アクティブユーザー**を掲げています。
  - オフラインの速さや、Apple純正より使いやすいという声が前面に出ています。

### 3. Superwhisper
- **料金**：
  - **無料枠あり**
  - 本格利用は有料サブスク中心
- **特徴**：
  - Mac / Windows / iPhone対応
  - あらゆるアプリで使える
  - カスタマイズ性が高い
  - ローカル処理も強い
  - 設定項目がかなり多く、玄人っぽい楽しさがある
- **評判**：
  - 高精度、オフラインで安心、多機能という点が好評
  - 一方で、機能が多くて少し複雑という印象を持つ人もいます

### 4. Wispr Flow
- **料金**：
  - **無料枠あり**
  - 有料プランあり
- **特徴**：
  - Mac / Windows / iPhone / Android対応
  - クロスプラットフォームが強い
  - 自然な書き直しや整形が得意
  - チーム機能もある
- **評判**：
  - 「速い」「精度が高い」「どのアプリでも自然に使える」という評価が強いです

### 5. Aqua Voice
- **料金**：
  - **無料トライアルあり**
  - その後は有料利用が中心
- **特徴**：
  - Mac / Windows対応
  - 文脈に応じた整形が強い
  - 技術文書・メール・書籍用途で高精度をうたう
  - 自動言語切替にも対応
- **評判**：
  - 「速い」「正確」「信頼できる」「サポートが良い」といった点が強く評価されています

### 6. MacWhisper
- **料金**：
  - ローカルモデルは無料で使える
  - クラウド転写やAI要約などの支援機能は有料
- **特徴**：
  - もともとは**音声ファイルの文字起こし**で有名
  - ローカル処理が強い
  - プライバシー重視
  - 近年は system-wide dictation も打ち出している
- **評判**：
  - 「コスパが良い」「ローカル処理が安心」という声が目立ちます
  - ただし、リアルタイム音声入力専用ツールというより、**文字起こし寄りの印象**を持つ人もいます

## ざっくり比較するとどうなるか

一言でまとめると、こんな感じです。

- **とにかく手軽に始める**なら Apple純正
- **多機能で玄人向け**なら Superwhisper
- **クロスプラットフォーム重視**なら Wispr Flow
- **整形の賢さ重視**なら Aqua Voice
- **ファイル文字起こし寄り**なら MacWhisper
- **無料・オフライン・Mac/iPhoneのバランス**で見ると Spokenly がかなり強い

ここ、けっこう面白いところです。  
AI音声入力界隈は、だいたい最後に**「精度は高いけど月額がかかる」**問題にぶつかります。  
便利なんだけど、毎月払うのか……となる。現代のサブスク沼、ぬかるみが深い。

## Spokenlyを無料で使う方法

Spokenlyは、**無料で使うだけならかなりシンプル**です。

### 手順
1. Mac あるいは iPhone に Spokenly を入れる  
2. **Local Models** を選ぶ  
3. 必要ならモデルをダウンロードする  
4. ショートカットキーを押して話す  
5. 話し終わったら離すと、カーソル位置に文字が入る

これだけです。

さらに、クラウドモデルを使いたい場合も、**自分のAPIキーを入れる方式（BYOK）**なら、Spokenly側の有料契約なしで使えます。  
つまり、Spokenlyに月額を払わず、必要なら自分でOpenAIやDeepgramなどのAPI利用分だけ負担する形です。

## なぜSpokenlyは無料で使えるのか

理由はわりと明快です。

Spokenlyは、**無料部分をローカル処理中心で構成している**からです。  
ローカルモデルを使うと、音声データは自分のMac上で処理され、Spokenly側に継続的な推論コストがほぼ発生しません。  
さらにBYOKでは、クラウド費用をユーザー自身のAPI利用に切り分けています。  
だから、**「アプリの器」は無料でも成立しやすい**わけです。

逆に、有料ツールの多くはクラウドで毎回高精度処理を回すため、どうしても運営側のコストが乗ります。  
そのため、月額課金になりやすいのです。

## 有料ツールとの違いは何か

無料のSpokenlyと、有料寄りの他ツールとの違いを雑に言うと、次の3点です。

### 1. サーバーを誰が負担するか
- **Spokenly無料**：自分のMacで処理、または自分のAPIキーを使う
- **有料ツール**：サービス側のクラウドを使うことが多い

### 2. 最初からどこまで整えてくれるか
- 有料ツールは、話した内容をそのまま文字にするだけでなく、**自然な文章・用途別フォーマット・文脈反映**まで一体でやってくれることが多いです
- Aqua VoiceやWispr Flowはこの路線が強めです

### 3. 使い勝手の“摩擦”の少なさ
- 有料ツールは、初期設定なしで「すぐ気持ちよく使える」方向に寄せていることが多い
- Spokenlyは無料でかなり強い反面、**ローカルモデルを選ぶ・必要ならAPIキーを入れる**といった、少しだけ自分で触る部分があります

## 結論：まず試すならSpokenlyがかなり良い

AI音声入力を試したい人にとって、最初の壁はだいたい  
**「月額を払うほど使うか、まだ分からない」**  
です。

その点、Spokenlyは

- ローカルモデルが**無料・無制限**
- **オフラインで使える**
- MacとiPhoneの両方で使える
- 必要ならあとでProやBYOKに進める

という構成なので、**入口としてかなり優秀**です。

もちろん、最終的に「もっと賢く整形してほしい」「チームで使いたい」「最初から全部お膳立てしてほしい」となれば、Wispr FlowやAqua Voice、Superwhisperのような有料ツールが候補に入ってきます。  
でも、まず最初の一本としては、Spokenlyはかなり筋がいい。  
無料でここまでできるなら、なかなかしたたかです。
